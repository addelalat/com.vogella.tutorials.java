== Exercise: Using OkHttp

	
This is an example for the usage of OkHttp in a standard Java program, but this library can also be used in Android applications.
This example demonstrates the usage of the API.
	
Create a new Java project called `com.vogella.java.library.okhttp`.
	
Download the OkHttp JAR and its requirements and add them to the build path of your project.
	
	
Afterwards create the following test class which is more or less taken for the http://square.github.io/okhttp/[OkHttp].
