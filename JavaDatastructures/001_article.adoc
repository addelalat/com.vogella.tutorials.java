List, Map and Stack implementations in Java. 
This article describes how to implement data structures (List Stack, Map) in Java.
The implementations in this articles are for demonstration and education purpose.
They do not try to be as efficient as the standard libraries and they are not intended to be an replacement for the standard libraries.
