JPA with Eclipselink.
This tutorial explains how to use EclipseLink, the reference implementation for the Java Persistence API (JPA). 
The usage of EclipseLink is demonstrated for stand-alone Java applications (outside the Java EE environment). 
The EclipseLink 2.3.X implementation is used for this tutorial.
